**三、 (15 分)** 设 $X(t) = Am(t)\cos(2\pi f_c t) - B\hat{m}(t)\sin(2\pi f_c t)$，其中 $m(t)$ 是零均值平稳基带随机过程，功率为 $P_m$，带宽为 $W$，$W \ll f_c$。$\hat{m}(t)$ 是 $m(t)$ 的希尔伯特变换。

**(1) 写出一组能使 $X(t)$ 平稳的系数 $A、B$ 的取值 (除全零外)，并证明之；**

解：
信号 $X(t)$ 可以表示为 $X(t) = X_c(t)\cos(2\pi f_c t) - X_s(t)\sin(2\pi f_c t)$，其中同相分量 $X_c(t) = Am(t)$，正交分量 $X_s(t) = B\hat{m}(t)$。· 
为了使带通信号 $X(t)$ 是宽义平稳 (WSS) 的，其同相分量 $X_c(t)$ 和正交分量 $X_s(t)$ 必须是联合宽义平稳的，并且它们的自相关函数和互相关函数满足以下条件：
1.  $R_{X_c}(\tau) = R_{X_s}(\tau)$
2.  $R_{X_c X_s}(\tau) = -R_{X_s X_c}(\tau)$

其中 $m(t)$ 是零均值平稳过程，功率为 $P_m = E[m^2(t)]$。其自相关函数为 $R_m(\tau) = E[m(t)m(t+\tau)]$。
$\hat{m}(t)$ 是 $m(t)$ 的希尔伯特变换。已知希尔伯特变换不改变信号的功率，且 $m(t)$ 与 $\hat{m}(t)$ 在同一时刻是正交的（对于零均值过程，即不相关）。
$E[\hat{m}^2(t)] = P_m$，所以 $R_{\hat{m}}(\tau) = R_m(\tau)$。
$E[m(t)\hat{m}(t+\tau)] = R_{m\hat{m}}(\tau) = \hat{R}_m(\tau)$ (即 $R_m(\tau)$ 的希尔伯特变换)。
$E[\hat{m}(t)m(t+\tau)] = R_{\hat{m}m}(\tau) = -\hat{R}_m(\tau)$。

现在计算 $X_c(t)$ 和 $X_s(t)$ 的相关函数：
$R_{X_c}(\tau) = E[Am(t)Am(t+\tau)] = A^2 R_m(\tau)$
$R_{X_s}(\tau) = E[B\hat{m}(t)B\hat{m}(t+\tau)] = B^2 R_{\hat{m}}(\tau) = B^2 R_m(\tau)$
$R_{X_c X_s}(\tau) = E[Am(t)B\hat{m}(t+\tau)] = AB R_{m\hat{m}}(\tau) = AB \hat{R}_m(\tau)$
$R_{X_s X_c}(\tau) = E[B\hat{m}(t)Am(t+\tau)] = AB R_{\hat{m}m}(\tau) = -AB \hat{R}_m(\tau)$

应用平稳条件：
1.  $R_{X_c}(\tau) = R_{X_s}(\tau) \implies A^2 R_m(\tau) = B^2 R_m(\tau)$。由于 $m(t)$ 不是恒为零的过程 ($P_m \ne 0$)，所以 $R_m(\tau)$ 不恒为零。因此，$A^2 = B^2$，即 $|A| = |B|$。
2.  $R_{X_c X_s}(\tau) = -R_{X_s X_c}(\tau) \implies AB \hat{R}_m(\tau) = -(-AB \hat{R}_m(\tau)) = AB \hat{R}_m(\tau)$。这个条件总是满足的，只要 $\hat{R}_m(\tau)$ 不恒为零（即 $m(t)$ 不是仅含直流或特定周期信号）。

因此，使 $X(t)$ 平稳的条件是 $|A|=|B|$。
一组满足条件的非全零系数可以是 $A=1, B=1$。
证明：若 $A=1, B=1$，则 $A^2=1, B^2=1$，满足 $A^2=B^2$。
(此时 $X(t) = m(t)\cos(2\pi f_c t) - \hat{m}(t)\sin(2\pi f_c t)$，这是一个下边带 (LSB) SSB 信号。)

**(2) 将 $X(t)$ 经过下图所示的系统后成为 $Y(t)$，假设低通滤波器的带宽等于 $m(t)$ 的带宽 $W$，写出 $Y(t)$ 的表达式；**
$X(t) \longrightarrow \underset{\otimes}{\longrightarrow} \text{LPF} \longrightarrow Y(t)$
$\qquad \qquad \uparrow 2\cos(2\pi f_c t + \theta)$

解：
相干解调过程：
令 $Z(t) = X(t) \cdot 2\cos(2\pi f_c t + \theta)$
$Z(t) = [Am(t)\cos(2\pi f_c t) - B\hat{m}(t)\sin(2\pi f_c t)] \cdot 2\cos(2\pi f_c t + \theta)$
使用积化和差公式：
$\cos\alpha \cos\beta = \frac{1}{2}[\cos(\alpha-\beta) + \cos(\alpha+\beta)]$
$\sin\alpha \cos\beta = \frac{1}{2}[\sin(\alpha+\beta) + \sin(\alpha-\beta)]$
$Z(t) = 2Am(t)\cos(2\pi f_c t)\cos(2\pi f_c t + \theta) - 2B\hat{m}(t)\sin(2\pi f_c t)\cos(2\pi f_c t + \theta)$
$= Am(t)[\cos(-\theta) + \cos(4\pi f_c t + \theta)] - B\hat{m}(t)[\sin(4\pi f_c t + \theta) + \sin(-\theta)]$
$= Am(t)[\cos\theta + \cos(4\pi f_c t + \theta)] - B\hat{m}(t)[\sin(4\pi f_c t + \theta) - \sin\theta]$
$= Am(t)\cos\theta + Am(t)\cos(4\pi f_c t + \theta) - B\hat{m}(t)\sin(4\pi f_c t + \theta) + B\hat{m}(t)\sin\theta$

低通滤波器 (LPF) 的带宽为 $W$。$m(t)$ 和 $\hat{m}(t)$ 的带宽均为 $W$。
项 $Am(t)\cos(4\pi f_c t + \theta)$ 和 $-B\hat{m}(t)\sin(4\pi f_c t + \theta)$ 是中心频率在 $2f_c$ 附近的信号。由于 $W \ll f_c$，这些高频分量将被 LPF 滤除。
因此，输出 $Y(t)$ 为：
$Y(t) = Am(t)\cos\theta + B\hat{m}(t)\sin\theta$

**(3) 若 $\theta$ 是常数，求 $Y(t)$ 的平均功率；**

解：
$Y(t)$ 的平均功率 $P_Y = E[Y^2(t)]$。
$P_Y = E[(Am(t)\cos\theta + B\hat{m}(t)\sin\theta)^2]$
$= E[A^2 m^2(t)\cos^2\theta + B^2 \hat{m}^2(t)\sin^2\theta + 2ABm(t)\hat{m}(t)\cos\theta\sin\theta]$
由于 $E[\cdot]$ 是线性算子，且 $\theta$ 是常数：
$P_Y = A^2\cos^2\theta E[m^2(t)] + B^2\sin^2\theta E[\hat{m}^2(t)] + 2AB\cos\theta\sin\theta E[m(t)\hat{m}(t)]$
已知 $E[m^2(t)] = P_m$ (m(t)的功率)。
$E[\hat{m}^2(t)] = P_m$ (希尔伯特变换不改变功率)。
$m(t)$ 是零均值过程，$\hat{m}(t)$ 也是零均值过程。$m(t)$ 和 $\hat{m}(t)$ 在同一时刻是正交的，即 $E[m(t)\hat{m}(t)] = 0$。
所以，$P_Y = A^2 P_m \cos^2\theta + B^2 P_m \sin^2\theta + 0$
$P_Y = P_m (A^2\cos^2\theta + B^2\sin^2\theta)$
如果采用 (1) 中使 $X(t)$ 平稳的条件，例如 $A=1, B=1$ (或更一般地 $A^2=B^2=K^2$ for some constant $K \ne 0$)：
若 $A^2=B^2=K^2$，则 $P_Y = P_m K^2(\cos^2\theta + \sin^2\theta) = K^2 P_m$。
例如，若 $A=1, B=1$, 则 $P_Y = P_m$。

**(4) 若 $\theta$ 是在 $[0, 2\pi]$ 上均匀分布的随机变量，求 $Y(t)$ 的平均功率。**

解：
当 $\theta$ 是随机变量时，$Y(t)$ 的平均功率需要对 $m(t)$ 和 $\theta$ 进行联合平均。
$P_Y = E_{m,\theta}[Y^2(t)] = E_\theta[E_m[Y^2(t) | \theta]]$
内部期望 $E_m[Y^2(t) | \theta]$ 就是在 $\theta$ 给定情况下的功率，已在 (3) 中求得：
$E_m[Y^2(t) | \theta] = P_m (A^2\cos^2\theta + B^2\sin^2\theta)$
现在对 $\theta$ 求期望：
$P_Y = E_\theta[P_m (A^2\cos^2\theta + B^2\sin^2\theta)]$
$P_Y = P_m A^2 E_\theta[\cos^2\theta] + P_m B^2 E_\theta[\sin^2\theta]$
$\theta$ 在 $[0, 2\pi]$ 上均匀分布，其概率密度函数 $f_\theta(\alpha) = \frac{1}{2\pi}$ for $\alpha \in [0, 2\pi]$。
$E_\theta[\cos^2\theta] = \int_0^{2\pi} \cos^2\alpha \frac{1}{2\pi} d\alpha = \frac{1}{2\pi} \int_0^{2\pi} \frac{1+\cos(2\alpha)}{2} d\alpha = \frac{1}{4\pi} [\alpha + \frac{1}{2}\sin(2\alpha)]_0^{2\pi} = \frac{1}{4\pi} (2\pi) = \frac{1}{2}$
$E_\theta[\sin^2\theta] = \int_0^{2\pi} \sin^2\alpha \frac{1}{2\pi} d\alpha = \frac{1}{2\pi} \int_0^{2\pi} \frac{1-\cos(2\alpha)}{2} d\alpha = \frac{1}{4\pi} [\alpha - \frac{1}{2}\sin(2\alpha)]_0^{2\pi} = \frac{1}{4\pi} (2\pi) = \frac{1}{2}$
所以，$P_Y = P_m A^2 (\frac{1}{2}) + P_m B^2 (\frac{1}{2}) = \frac{P_m}{2}(A^2+B^2)$
如果采用 (1) 中使 $X(t)$ 平稳的条件，例如 $A^2=B^2=K^2$：
则 $P_Y = \frac{P_m}{2}(K^2+K^2) = K^2 P_m$。
例如，若 $A=1, B=1$, 则 $P_Y = P_m$。

这表明，如果输入信号 $X(t)$ 是平稳的 (即 $|A|=|B|=K$)，则解调后低通输出信号 $Y(t)$ 的平均功率为 $K^2 P_m$，无论相位 $\theta$ 是固定常数还是在 $[0, 2\pi]$ 上均匀分布的随机变量。
